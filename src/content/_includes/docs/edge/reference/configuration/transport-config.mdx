ThingsBoard Edge runs the same transport services as the ThingsBoard Community Edition server — MQTT, HTTP, CoAP, LwM2M, and SNMP. The environment variables that configure each transport are identical between Edge and CE.

Use the CE transport configuration references below to look up specific variables. Set them in `/etc/tb-edge/conf/tb-edge.conf` (Linux) or as container environment variables (Docker / Kubernetes). For instructions on applying configuration changes, see <DocLink product={props.product === Products.EDGE_PE ? Products.EDGE_PE : Products.EDGE} path="reference/configuration/how-to-change-config" target="_blank">How to change configuration</DocLink>.

## MQTT transport

Controls the MQTT broker — port, TLS, session limits, queue settings, and keep-alive tuning.

**Default ports:** 1883 (plain), 8883 (TLS)

For the full variable list, see <DocLink product={Products.CE} path="reference/configuration/mqtt-transport-config" target="_blank">MQTT Transport Configuration</DocLink>.

## HTTP transport

Controls the HTTP API server — port, TLS, request size limits, and session timeouts.

**Default ports:** 8080 (plain), 443 (TLS)

For the full variable list, see <DocLink product={Products.CE} path="reference/configuration/http-transport-config" target="_blank">HTTP Transport Configuration</DocLink>.

## CoAP transport

Controls the CoAP server — UDP port, DTLS, and message timeout settings.

**Default ports:** 5683 UDP (plain), 5684 UDP (DTLS)

For the full variable list, see <DocLink product={Products.CE} path="reference/configuration/coap-transport-config" target="_blank">CoAP Transport Configuration</DocLink>.

## LwM2M transport

Controls the LwM2M server — registration, firmware update, and bootstrap settings.

**Default ports:** 5685 CoAP, 5686 CoAP/DTLS, 5687 TCP, 5688 TCP/TLS

For the full variable list, see <DocLink product={Products.CE} path="reference/configuration/lwm2m-transport-config" target="_blank">LwM2M Transport Configuration</DocLink>.

## SNMP transport

Controls the SNMP polling engine — listen address, thread pool, and request timeout.

**Default port:** 161 UDP

For the full variable list, see <DocLink product={Products.CE} path="reference/configuration/snmp-transport-config" target="_blank">SNMP Transport Configuration</DocLink>.

<Aside type="note">
  ThingsBoard Edge does not expose the JS Executor or Version Control executor services. Those configuration pages apply to the ThingsBoard server only.
</Aside>
