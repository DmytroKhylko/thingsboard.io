The IoT Gateway is in active development. The following features are planned for upcoming releases:

- **DLMS/COSEM Connector** — support for the DLMS/COSEM protocol used in smart metering and energy devices.
- **GPIO support** — direct control and monitoring of GPIO pins on supported hardware.

For the latest release notes and version history, see the
[GitHub releases page](https://github.com/thingsboard/thingsboard-gateway/releases).
