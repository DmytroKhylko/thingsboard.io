Configure an alarm details dashboard to open when a user taps an alarm on the alarms screen.
The dashboard is set per alarm rule inside the device profile.

## Configure alarm details dashboard

<Steps>

1. Go to **Device profiles** in the left main menu.
2. Click the device profile you want to modify.
3. In the device profile details panel, navigate to the **Alarm rules** tab.
4. Click the **Edit** (pencil) button.
5. In the **Create alarm rule** form, select the desired dashboard in the **Mobile dashboard** field.
6. Click **Apply changes**.

</Steps>

  The alarm dashboard configuration applies only to alarms generated after the configuration is saved, not to existing alarms.
</Aside>
