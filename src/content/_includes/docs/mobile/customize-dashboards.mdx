ThingsBoard lets you control how dashboards appear on the mobile app's home screen — set a cover image,
define the display order, hide specific dashboards, and rearrange the navigation bar — all from the dashboard details form.

## Dashboard image

Upload a cover image that the mobile app displays on the dashboard tile in the home screen list.

<Steps>

{props.product === Products.MOBILE_PE ? (
  <ol>

    <li>Open the target dashboard group.</li>
    <li>Click the dashboard you want to modify.</li>
    <li>In the dashboard details panel, click the <strong>Edit</strong> (pencil) button.</li>

## Hide dashboard in mobile app

You can hide specific dashboards so they do not appear on the mobile app home screen.

<Steps>

{props.product === Products.MOBILE_PE ? (
  <ol>
    <li>Go to <strong>Dashboard groups</strong> in the left main menu.</li>
    <li>Open the target dashboard group.</li>
    <li>Click the dashboard you want to hide.</li>
    <li>In the dashboard details panel, click the <strong>Edit</strong> (pencil) button.</li>
    <li>Enable the <strong>Hide dashboard in mobile application</strong> checkbox.</li>
    <li>Click <strong>Apply changes</strong>.</li>
  </ol>
) : (
  <ol>
    <li>Go to <strong>Dashboards</strong> in the left main menu.</li>
