ThingsBoard lets you configure a custom icon for each device type by uploading an image to the device profile.
The mobile app uses this image to represent devices of that profile on the **Devices** screen.

## Device profile image

<Steps>

1. Go to **Device profiles** in the left main menu.
2. Click the device profile you want to modify.
3. In the device profile details panel, click the **Edit** (pencil) button.
4. Upload the desired image to the **Device profile image** field.
5. Click **Apply changes**.

</Steps>
