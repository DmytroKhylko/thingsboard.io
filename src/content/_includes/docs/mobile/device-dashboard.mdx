Configure a device details dashboard to open when a user taps a device of a specific profile in the mobile app.
The dashboard is set per device profile.

## Configure device details dashboard

<Steps>

1. Go to **Device profiles** in the left main menu.
2. Click the device profile you want to modify.
3. In the device profile details panel, click the **Edit** (pencil) button.
4. Select the desired dashboard in the **Mobile dashboard** field.
5. Click **Apply changes**.

</Steps>
