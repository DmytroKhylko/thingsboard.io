
Self-registration allows new users to create accounts independently, directly through the mobile app. New users can sign up and access the platform with predefined permissions through a registration form.

The Tenant Administrator configures this feature through the <a href="/docs/pe/user-guide/mobile-app-center/">Mobile center</a>.
