Publish telemetry by sending a CoAP `POST` request. See <DocLink product={props.product} path='reference/coap-api/getting-connected'>Getting Connected</DocLink> for credential types and connection details.

## Endpoints

| Credential type | URL |
|-----------------|-----|
| Access Token | <HostInline product={props.product} code="coap(s)://{COAP_HOST_LABEL}:5683/api/v1/$ACCESS_TOKEN/telemetry" /> |
| X.509 Certificate | <HostInline product={props.product} code="coaps://{COAP_HOST_LABEL}/api/v1/telemetry" /> |

## Payload Formats

### Simple Key-Value

A flat JSON object of key-value pairs. ThingsBoard assigns the server receive time as the timestamp.

```json
{"temperature": 42, "humidity": 73}
```

**Access Token:**

<HostCode product={props.product} lang="bash" code={`coap-client -v 6 -m POST -t "application/json" -e '{"temperature":42,"humidity":73}' "coap://{COAP_HOST}:5683/api/v1/$ACCESS_TOKEN/telemetry"`} />

**X.509 Certificate:**

<HostCode product={props.product} lang="bash" code={`coap-client-openssl -v 6 -c cert.pem -j key.pem -m POST -t "application/json" -e '{"temperature":42,"humidity":73}' "coaps://{COAP_HOST}/api/v1/telemetry"`} />

### With Client-Side Timestamp

Include `ts` (Unix milliseconds) when the device has a reliable clock and you want the exact measurement time stored instead of the server receive time.

```json
{"ts": 1451649600512, "values": {"temperature": 42, "humidity": 73}}
```

**Access Token:**

<HostCode product={props.product} lang="bash" code={`coap-client -v 6 -m POST -t "application/json" -e '{"ts":1451649600512,"values":{"temperature":42,"humidity":73}}' "coap://{COAP_HOST}:5683/api/v1/$ACCESS_TOKEN/telemetry"`} />

`1451649600512` is a Unix timestamp in milliseconds — corresponds to `Fri, 01 Jan 2016 12:00:00.512 GMT`.

### Array of Objects with Client-Side Timestamps

Send a batch of readings in a single request, each with its own timestamp. Use this to flush buffered measurements captured while offline.

```json
[
  {"ts": 1451649600512, "values": {"temperature": 42, "humidity": 73}},
  {"ts": 1451649601000, "values": {"temperature": 43, "humidity": 72}}
]
```

**Access Token:**

<HostCode product={props.product} lang="bash" code={`coap-client -v 6 -m POST -t "application/json" -e '[{"ts":1451649600512,"values":{"temperature":42,"humidity":73}},{"ts":1451649601000,"values":{"temperature":43,"humidity":72}}]' "coap://{COAP_HOST}:5683/api/v1/$ACCESS_TOKEN/telemetry"`} />
