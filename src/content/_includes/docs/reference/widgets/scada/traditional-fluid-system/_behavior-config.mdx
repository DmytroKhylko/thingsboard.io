To configure a parameter, click its edit icon. Each parameter has two fields:

- **Action** — determines how the widget reads the value:

    | Action | Description |
    |---|---|
    | **Do nothing** | Uses the static default value. No data is read from the device. |
    | **Execute RPC** | Calls an RPC method on the device and maps the result to the parameter. |
    | **Get attribute** | Reads a device attribute value. |
    | **Get time series** | Reads the latest telemetry value for the given key. |
    | **Get alarm status** | Maps alarm severity to the parameter value. |
    | **Get dashboard state id** | Uses the current dashboard state ID. |
    | **Get dashboard state object** | Uses the current dashboard state object. |

- **Value** — the static default used when **Action** is set to **Do nothing**.
