The QR code feature streamlines the authorization process in the ThingsBoard mobile application — users can scan a code to download the app or log in instantly, without entering credentials manually.

## Scan to download the mobile app

If the ThingsBoard Mobile Application has not been installed yet, scanning the QR code will redirect the user to the application marketplace (available on [Google Play](https://play.google.com/store/apps/details?id=org.thingsboard.cloud) and [App Store](https://apps.apple.com/us/app/thingsboard-cloud/id6499209395)).

Two types of application are available:

**Default** — uses the officially published **ThingsBoard Cloud** mobile application (available on [Google Play](https://play.google.com/store/apps/details?id=org.thingsboard.cloud) and [App Store](https://apps.apple.com/us/app/thingsboard-cloud/id6499209395)).

<Aside type="note">
  The ThingsBoard Cloud mobile application includes the capability to switch its server host, allowing it to work with your ThingsBoard instance. Select the "Login with scan QR code" option on the mobile app's login page, and the application will automatically switch its server host to your ThingsBoard platform host.
</Aside>

## Mobile app QR code widget on dashboards

You can also add the **Mobile app QR code** widget to your <DocLink product={props.product} path="user-guide/dashboards">dashboards</DocLink> to simplify the login process for customers on their mobile devices.

<Steps>
1. Open the dashboard in **edit mode**.
2. Click **Add widget**.
3. Search for the **Mobile app QR code** widget.
4. Configure the widget settings.
5. Save the dashboard.
</Steps>
