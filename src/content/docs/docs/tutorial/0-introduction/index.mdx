<Lede> This guide introduces the core ThingsBoard concepts and helps you get started with IoT data in just a few minutes. 🚀 </Lede>

By completing it, you will learn how to:
- Connect a device to ThingsBoard
- Publish telemetry data
- Visualize data on interactive dashboards
- Configure alarms and notifications
- Add a customer and securely share data with them

:::tip
At the end of each page, you will find an interactive checkbox to track your progress in the training materials, along with navigation buttons.  
Using the tracker is optional, but it can help you remember where you left off and mark topics for later review.

All data is stored locally in your browser and is not sent to or stored by ThingsBoard.
:::

---

## What do I need to do to get started?

You need ThingsBoard server up and running.  
For a fast start, you can use **ThingsBoard Cloud**. Simply follow the Getting Started guide for your region: [North America](/docs/paas/getting-started-guides/helloworld/) or [Europe](/docs/paas/eu/getting-started-guides/helloworld/).

Alternatively, you can deploy ThingsBoard on your own infrastructure using our guides for [Windows (Docker)](/docs/user-guide/install/docker-windows/) or [Linux / MacOS (Docker)](/docs/user-guide/install/docker/).

---

## Log in to ThingsBoard

Open the ThingsBoard web interface. On the login page, you can:
- **Sign up** using Google, GitHub, Apple, or your email.
- **Sign in** using existing account credentials.

After successful authentication, you will be redirected to the ThingsBoard interface.  
All further steps in this guide are performed at the **Tenant** level.

:::note
Available login options and branding may differ between ThingsBoard Cloud and self-hosted deployments.
:::
